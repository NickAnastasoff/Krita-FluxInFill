Background Remover BriaAI v1.2

A simple plugin for Krita that quickly and automatically infills transparent pixels with replicates api

Usage:

  1. You must first create a replicate account and an API key on https://replicate.com/. Copy and paste the API key in the API key field.
  2. Erase the part of the image you want to inpaint. It will then create a new layer with the whole generation.

I hope you enjoy the power and ease of infill in Krita!

Visit https://github.com/ for the most recent version.
